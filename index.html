Fase 1 · Desafío de Productividad Grupal

Inicio: 22 de junio

Actividad grupal por mesas para registrar tareas entregadas a tiempo y reforzar la productividad con competencia sana.

1 diapositivaUna lámina clara para explicar la actividad de la fase.

Fase 2 · La Isla de la Procrastinación

Inicio: 1 de julio

Dinámica grupal para identificar distractores, reconocer debilidades y construir una ruta de escape con acciones concretas.

1 diapositivaUna lámina clara para presentar la dinámica de la isla.
2 guionesGuion para 8vo y guion para 7mo, con distribución distinta de líneas.
Materiales para imprimirPlantilla, instrucciones o recurso físico necesario para la dinámica.

Fase 3 · Plan Maestro de Rutina

2 guionesGuion para 8vo y guion para 7mo, ajustados a cada grupo.
Materiales para imprimirFicha o recurso físico necesario para que el estudiante trabaje.

Añadir material

Registra un recurso usando enlace, archivo adjunto o ambas opciones. El archivo puede ser imagen, foto, PDF, Word, PowerPoint u otro documento.

El panel guarda los registros en este navegador. Si adjuntas archivos, también se guardan dentro del respaldo JSON; para archivos muy pesados conviene subirlos a Drive/Canva y pegar el enlace.

Puedes adjuntar un archivo local. Para compartirlo con todos, exporta el respaldo JSON o usa también un enlace de Drive/Canva.
